------------------------------------------------------------------------
Este proyecto consiste en evaluar si una contraseña dada es correcta o no.
El proyecto contiene una clase con un solo metodo que lee entradas del teclado
y evalúa si las entradas son iguales a la contrasela o no. Tambien cuenta con
un main que ejecuta una prueba del método.
------------------------------------------------------------------------

PROJECT TITLE: Contraseña
PURPOSE OF PROJECT: Evaluar si una contraseña dada es correcta o no
VERSION or DATE: 1.1
HOW TO START THIS PROJECT: Ejecutando el main de la clase Contraseña
